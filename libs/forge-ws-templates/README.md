## forge-ws-templates
[![npm version](https://badge.fury.io/js/%40creditchain%2Fforge-ws-templates.svg)](https://www.npmjs.com/package/@creditchain/forge-ws-templates)


`@creditchain/forge-ws-templates` is used to create a workspace using different templates on Forge IDE.

### Installation

`@creditchain/forge-ws-templates` is an NPM package and can be installed using NPM as:

`yarn add @creditchain/forge-ws-templates`
